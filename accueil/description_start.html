Description du projet

Par Florent, Jonas et Loïc le 13/03/2019 à 17:15

Dans le cadre du cours de Génie Logiciel, nous avons pour projet de faire un jeu vidéo en 2D. Nous avons donc décidé de faire un jeu vidéo ayant pour thème la foire du Valais.

Le but de notre jeu, sera de passer une journée complète, du matin à l’ouverture jusqu’au soir à la fermeture, en allant aux différents stands qui vendent des bières ou spiritueux sans faire de coma éthylique.

Durant la partie, le personnage pourra se déplacer sur la carte représentant le CERM à Martigny avec les différents stands placés à leur endroit réel. Le taux d’alcool dans le sang modifiera au fur et à mesure les déplacements du personnage. En effet, à partir d’un certain seuil d’alcoolémie, le personnage se déplacera en tanguant.

Pour marquer un maximum de points, le joueur devra avoir le plus d'alcool dans le sang, sans pour autant avoir sa vessie trop pleine. En effet, le score est calculé en fonction du taux d'alcoolémie et du remplissage de la vessie du joueur.

Le code source de l'application est disponible sur le github du projet.
